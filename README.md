# Housing Dataset - Feature Engineering

## Overview
Performed data preprocessing and feature engineering on the Housing dataset using Pandas and Scikit-Learn.

## Tasks Performed
- One-Hot Encoding of categorical features
- Feature Scaling using StandardScaler
- Created interaction features:
  - area_bedrooms
  - area_bathrooms
  - bedrooms_bathrooms

## Tools Used
- Python
- Pandas
- Scikit-Learn

## Output
Generated a machine learning-ready dataset:
`Housing_Feature_Engineered.csv`

## Conclusion
The dataset was transformed by encoding categorical variables, scaling numerical features, and creating interaction terms to improve model performance.
